# 💰 Kurmatik - Modern Finans Uygulaması

**Real-time Currency & Gold Price Converter with Smart Alerts**

Kurmatik, gerçek zamanlı döviz kurları ve altın fiyatları sunan, akıllı uyarı sistemi ile donatılmış modern bir React Native uygulamasıdır.

## 🚀 Özellikler

### 💱 Döviz Çevirici
- Desteklenen Para Birimleri: USD, EUR, TRY, GBP, QAR, IRR, IQD, RUB
- Gerçek Zamanlı Kurlar: Yahoo Finance API entegrasyonu
- Çapraz Kurlar: Herhangi bir para biriminden diğerine çevrim

### 🥇 Altın Fiyatları
- Gram, Çeyrek, Yarım, Tam, Cumhuriyet, Ons Altın
- XAU Entegrasyonu: Ons bazlı uluslararası fiyatlar

### 🔔 Akıllı Uyarı Sistemi
- Fiyat Uyarıları: Kur ve altın fiyatları için hedef değer
- Kalıcı Depolama: AsyncStorage ile kaydetme
- Real-time Monitoring: Otomatik fiyat kontrolü

### 🎨 Modern UI/UX
- Responsive Tasarım: Mobil ve web uyumlu
- Dark Mode: Profesyonel görünüm
- Swipe Navigation: Sayfa geçişi

### 🔧 Teknik Özellikler
- Free API: Tamamen ücretsiz Yahoo Finance API
- No Quota: Kota sınırlaması yok
- Offline Support: nternet yokken static değerler

## 📡 API Kaynakları

- **Yahoo Finance**: Döviz kurları
- **Truncgil**: Altın fiyatları
- **TradingView**: Grafik widget'ları
